Student Management System

A full stack Student Management System built using:

Frontend:
- React

Backend:
- Node.js
- Express

Database:
- SQL Server

Features:
- Add Student
- Edit Student
- Delete Student
- View Students

Live Demo:
[https://your-netlify-link.netlify.app](https://sernstdmanagement.netlify.app/)

Frontend deployed on Netlify
Backend built with Node.js and Express
